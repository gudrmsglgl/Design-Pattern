## Chain Of Responsibility Pattern

> 하나이상의 체인형태로 구성된 객체에게 처리를 위임하고 <br> 
> 요청을 처리할 수 있는 객체를 만날 때까지 <br> 
> 다음 객체로 처리를 위임하는 패턴

### 📍 절차
- 처리 인터페이스 추출 
- 처리 세분화 구현 객체
    - [x] 다음 처리 객체를 파라미터로 받음
    - [x] 구현 시 처리할 수 있는 부분은 처리하고, 처리하지 못한 부분은 다음 객체로 처리 위임

### 실행 출력
```
money: 825 => 100원: 8개
money: 25 => 10원: 2개
money: 5 => 1원: 5개
```
